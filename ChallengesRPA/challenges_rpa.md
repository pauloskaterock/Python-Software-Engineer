Sites Especializados em Desafios RPA

1. RPA Challenge (rpachallenge.com)
O desafio clássico – você deve baixar uma planilha Excel e preencher um formulário web onde os campos mudam de posição a cada inserção. O grande desafio está em criar seletores robustos (XPath/CSS) que funcionem dinamicamente. No final, o site mostra seu tempo de execução – perfeito para você tentar bater seu próprio recorde.

Conceitos praticados: Selenium/Playwright, leitura de Excel com pandas/openpyxl, seletores dinâmicos.

1. BotsDNA (botsdna.com)
Criado por Sudheer Nimmagadda (MVP UiPath), este site tem desafios mais tricky. Um exemplo é o "Customer Locator", onde você precisa extrair uma tabela que muda de estrutura a cada carregamento da página, armazenar em Excel, filtrar os dados e fazer manipulações avançadas.

Conceitos praticados: Seletores para tabelas dinâmicas, pandas para manipulação de dados, lógica condicional.

💻 Plataformas de Código e Repositórios
3. GitHub - Repositórios RPA Challenge
Buscar por rpa-challenge no GitHub traz dezenas de soluções prontas em Python que você pode estudar, clonar e modificar. É uma mina de ouro para ver como outros engenheiros resolveram os mesmos problemas.

Como usar: Não copie – analise, entenda a lógica e depois recrie do zero com sua abordagem.

1. GitHub - Robocorp Examples
A Robocorp tem frameworks excelentes para RPA com Python. Busque por robocorp + challenge no GitHub para encontrar exemplos profissionais com boas práticas como máquina de estados (emulando o ReFramework do UiPath).

🏆 Competições e Desafios com Dinheiro
5. Topcoder - Python Automation Robot Framework
A Topcoder realiza competições com três níveis de dificuldade (Easy, Medium, Hard) valendo de 250 a 1000 pontos. Os prêmios incluem camisetas e bônus de $50 para primeiros envios.

Tecnologias exigidas: Python 3.9, Robot Framework, Selenium.

1. Upwork - Projetos RPA Reais
Embora não seja um site de "desafios", filtrar por RPA + Python no Upwork mostra demandas reais do mercado. Pegue descrições de projetos reais e tente implementar como treino.

📚 Outras Plataformas Complementares
7. HackerRank - Python Skill Tests
Ótimo para fortalecer fundamentos de Python que serão usados nos scripts RPA: manipulação de strings, expressões regulares, estruturas de dados.

1. LeetCode - Database e Pandas
Automação quase sempre envolve dados. Os problemas de Pandas no LeetCode preparam você para extrações, transformações e consolidações de planilhas.

2. Codewars - Python Kata
Problemas de lógica que treinam o raciocínio para criação de seletores e fluxos condicionais complexos.

3. TechVyapy - Code Challenges
Lançaram recentemente 45 desafios de código em diferentes níveis, com foco em automação
